简单的 CurseForge API

这是curseforge.com的一个简单API。
它主要使用与 CurseForge 网站相同的URL方案。你可以把这个API想象成 CurseForge 的 JSON 浏览。
API 会尝试根据请求直接获取数据，但所有请求都会缓存 1 小时。建议使用客户端缓存。



当前支持的方法

列出项目

/{游戏}/{项目}[?category={分类}][&filter-game-version={游戏版本}][&filter-sort={领域}][&page={页码}]

/minecraft/mc-mods

搜索项目

/{游戏}/{项目}/search?search={搜索项}

/minecraft/mc-mods/search?search=worldedit

项目信息

/{游戏}/{项目}/{slug}

/minecraft/mc-mods/worldedit

项目关系

/{游戏}/{项目}/{slug}/relations/{关系}[?filter={领域}][&page={页码}]

/minecraft/mc-mods/journeymap/relations/dependencies

项目文件列表

/{游戏}/{项目}/{slug}/files[?sort={领域}][&page={页码}][&filter-game-version={版本ID}][&filter-game-version-name={版本名称}]

/minecraft/mc-mods/worldedit/files

文件信息

/{游戏}/{项目}/{slug}/files/{文件ID}

/minecraft/mc-mods/worldedit/files/2725648

下载信息

/{游戏}/{项目}/{slug}/download/{文件ID}

/minecraft/mc-mods/worldedit/download/2725648

分类列表

/other/{游戏}/{项目}/categories

/other/minecraft/mc-mods/categories

列出游戏版本

/other/{游戏}/{项目}/versions

/other/minecraft/mc-mods/versions
